O que são eventos de mouse e teclado em JavaScript?

  • Interação do usuário
  • Eventos
  • Ações feitas pelo usuário
  • Próxima ação
  • Elementos sejam alterados

  • Ex.: passar o mouse sobre um botão, um bloco de texto, uma única palavra, um ícone ou uma imagem.

Tipos de eventos de mouse e teclado:

  • onclick: o elemento é clicado pelo usuário
  • onkeypress: o usuário pressiona uma tecla sobre o elemento
  • onmouseover: define ação quando o usuário passa o mouse sobre o elemento
  • onmouseout: define ação quando o usuário retira o mouse sobre o elemento

Há outros tipos de eventos que envolvem mouse e teclado:

  • onblur: remove o foco do elemento
  • onfocus: o elemento é focado
  • onchange: muda o valor do elemento
  • onload: carrega o elemento por completo
  • onsubmit: define ação ao enviar um formulário

Somente mouse: onclick, onmousedown, onmouseup

O que é uma função de callback (utilizada no Event Listener)?

É uma função que só entra em operação após o processamento de outra. Ou seja, ela fica aguardando outra função ser executada para daí sim começar a rodar.

Exemplo:

const button = document.querySelector(“button”);
button.addEventListener(“click”, () => {
       console.log(“UHHUUUU eu sou um botão!!!”);
});

Qual a diferença de um Event Listener de atributos HTML (como onclick)?

Event Listener

  • Vários manipuladores
  • Remoção com 'removeEventListener()'
  • Legibilidade e manutenibilidade

Atributos HTML

  • Difícil manutenção no longo prazo
  • Oposto à SoC (Separation of Concerns)
  • Permite um único manipulador por evento e elemento
  • Para remover basta deixar como null ou false

O que é Event Bubbling em JS?

  • Interações acionam eventos
  • Clique em um elemento
  • Propagação em "bolha"
  • Todos os ancestrais

Diferença:

  • Propagação de baixo para cima

O que faz a função 'event.stopPropagation'?

  • Cessa a propagação
  • Pode ser colocada no elemento onde ocorre o evento
  • Os ancestrais não sofrem o efeito
